# 🛍️ ELVO – Fashion E-commerce Website

A full-stack e-commerce website where users can browse products, add them to cart, and place orders. It also includes an admin panel to manage products and orders.

---

## 🚀 Features
- Browse and view products  
- Add to cart and manage items  
- Place orders  
- Admin panel for product and order management  
- Responsive design  

---

## 🧑‍💻 Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB
